AI Career Intelligence OS 🧠

🚀 **Live Demo:** https://oqnuekybkp8trb8ei3kbj5.streamlit.app/

While building AI projects, I noticed that students and job seekers often need multiple tools for resume analysis, interview preparation, research, and skill assessment. So I challenged myself to build a single platform that could do all of these things in one place.

After a lot of learning, debugging, testing, and Tea☕, I finally built it.

### What can it do?

📄 **Document Brain**
Upload a resume or document and chat with it using AI-powered RAG.

🎯 **Resume Analyzer**
Get ATS scores, feedback, and suggestions to improve your resume.

📈 **Skill Gap Analyzer**
Find the skills you're missing for your target role and get insights on what to learn next.

🔍 **AI Research Agent**
Generate detailed research reports on any topic along with visualizations and downloadable PDFs.

💬 **Document Chat**
Have a conversation with your uploaded documents and get contextual answers instantly.

🎤 **AI Interview Coach**
Practice interviews and receive AI-generated feedback to improve your performance.

### Tech Stack

Python • Streamlit • Groq API • RAG • Vector Embeddings • Pandas • ReportLab • Generative AI

### What I learned from this project

* Building real-world AI applications
* Working with LLMs and RAG systems
* Prompt Engineering
* Streamlit deployment
* Document processing pipelines
* Creating complete end-to-end products instead of just models

This project pushed me beyond simply training models and helped me understand how AI products are actually built and deployed for users.
